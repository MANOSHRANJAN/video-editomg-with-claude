# video-edit

Style-driven short-form vertical video pipeline.

```
References (clean girl / luxury beauty)
        │
        ▼
┌──────────────────────┐
│  analyze.py          │  PySceneDetect + cv2 → style.json
│  (Python + uv venv)  │  (pacing, palette, keywords)
└──────────────────────┘
        │
        ▼
┌──────────────────────┐
│  fetch-broll.mjs     │  Pexels API (free) → public/broll/
│  generate-plan.mjs   │  + your A-roll → src/edit-plan.json
└──────────────────────┘
        │
        ▼
┌──────────────────────┐     ┌──────────────────┐     ┌──────────────────┐
│  Remotion (composer) │  +  │  Hyperframes     │  +  │  Palmier (MCP)   │
│  React timeline      │     │  HTML/CSS/GSAP   │     │  visual review   │
│  npx remotion studio │     │  npm run dev     │     │  desktop app     │
└──────────────────────┘     └──────────────────┘     └──────────────────┘
        │
        ▼
       MP4
```

## Stack

| Tool | Role | Status |
|---|---|---|
| **Remotion** | Code → MP4 timeline composer (React) | Installed, `npx remotion studio` |
| **Hyperframes** | Code → MP4 scene/title generator (HTML/CSS) | Scaffolded in `hyperframes-scenes/` |
| **Palmier Pro** | Visual editor + MCP-driven timeline | Installed at `/Applications`, MCP connected |
| **Pexels API** | Free real B-roll footage | API key in `.env`, smoke-tested |
| **PySceneDetect + cv2** | Style fingerprint extraction | `.venv/` (Python 3.12 via uv) |
| **claude-watch** | Skill for Claude to watch videos | Cloned to `~/.claude/skills/watch/` (restart needed) |

## Style fingerprint (current)

From the 6 reference videos in `style-analysis/references/`:

- **Pacing**: median shot 2.31s, avg 3.23s, ~0.31 cuts/sec
- **Look**: bright (0.65), low saturation (0.10), neutral palette `#f8f8f8 #b0afae #826e53 #161514`
- **Keywords**: `luxury`, `clean`, `girl`, `makeup`
- **Format**: 720x1280 vertical (output target: 1080x1920)

## Workflow

```bash
# 1. (Re-)analyze reference videos → style.json
./.venv/bin/python style-analysis/analyze.py

# 2. Fetch Pexels B-roll using style keywords
node scripts/fetch-broll.mjs
#    (or: node scripts/fetch-broll.mjs your custom keywords)

# 3. Drop your A-roll footage into public/aroll/

# 4. Generate edit-plan.json (uses style pacing on your footage)
node scripts/generate-plan.mjs --duration 20

# 5. Preview / tweak
npx remotion studio

# 6. Render
npx remotion render ReelEdit out/reel.mp4
```

## Hyperframes scenes

For animated title cards / data scenes / motion graphics:

```bash
cd hyperframes-scenes
npm run dev          # browser preview
npm run render       # MP4 → drop into ../public/broll/
```

Then `node scripts/import-broll.mjs` to wire the new MP4 into the Remotion plan.
